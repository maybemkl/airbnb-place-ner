# airbnb-place-ner

This repository contains the data and code to train and validate the models in the paper ["Toward a Critical Toponymy Framework for Named Entity Recognition: A Case Study of Airbnb in New York City"](https://aclanthology.org/2023.emnlp-main.284/), presented at EMNLP 2023. The benchmark DistilRoBERTa-CRF model from the paper can be downloaded [here]().
